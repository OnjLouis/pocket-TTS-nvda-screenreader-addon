Pocket TTS for NVDA

This is a personal maintained fork of the Pocket TTS NVDA add-on for NVDA 2026.1 and later. Pocket TTS provides neural English voices for NVDA using the Pocket TTS ONNX model. It includes bundled voices and can create new voice embeddings from short WAV or MP3 samples.

Using Pocket TTS

Select Pocket TTS from NVDA's synthesizer dialog. Choose a voice from NVDA's speech settings or synth settings ring.

Voice Manager

Open NVDA Settings and choose Pocket TTS Voice Manager to add, rename, or remove voices.

  • Convert audio file creates a new voice embedding from the first 30 seconds of a WAV or MP3 file.
  • Select voice chooses an installed voice.
  • Display name edits the file name used for the selected voice.
  • Rename voice file applies the new display name.
  • Remove selected voice deletes the selected voice after confirmation.

Speech Settings

Flow steps controls how many model refinement passes are used while generating speech. Use 1 for the fastest response. Higher values may improve stability or quality for some voices, but can make speech slower.

  • Marked voices as English so NVDA can process characters and symbols more consistently.
  • Improved the voice manager so converting a new voice no longer blocks NVDA.
